# Bunnyland GitHub defaults

This repository provides the shared issue forms and pull request template for Bunnyland
repositories owned by `thalismind`.

GitHub applies these files to repositories that do not define a corresponding local
template. Keep repository-specific test commands and implementation details in each
repository's own contributor documentation rather than copying templates between repos.

Repository-local overrides are appropriate when the questions or verification contract
materially differ. Server and Web keep specialized issue and pull request templates for
that reason. Do not add pointer templates or copies of these generic defaults; repositories
without a real specialization should inherit this repository directly.
